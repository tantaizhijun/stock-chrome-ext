# 🍭 使用说明


    1. 使用Git下载代码库到本地, (或者下载压缩包到本地解压文件夹);

    2.打开谷歌浏览器, 打开: 设置>更多工具>扩展程序 页面;

    3.打开"开发者模式", 选择"加载已解压的扩展程序", 然后选择刚才解压的文件夹即可.

    4.点击扩展程序图标,把图钉选中, 程序小图标就显示在扩展栏里面了.
